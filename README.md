# Lettermint mail driver for Flarum 2.0

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/riclep/flarum-lettermint-mail.svg)](https://packagist.org/packages/riclep/flarum-lettermint-mail) [![Total Downloads](https://img.shields.io/packagist/dt/riclep/flarum-lettermint-mail.svg)](https://packagist.org/packages/riclep/flarum-lettermint-mail)

A [Flarum](http://flarum.org) extension to add support for [Lettermint](https://lettermint.co) mail.

Lettermint is an EU based transactional email provider that takes privacy seriously and offers a range of features to help you send and track your emails.

## Installation

```sh
composer require riclep/flarum-lettermint-mail
```

Once installed, you can enable this extension in the Flarum admin panel then configure it as your mail driver in Email core configuration.
